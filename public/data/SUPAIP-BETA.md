# SUP AIP BETA

`public/data/supaip-beta.geojson` est une couverture pilote partielle destinée à valider l'affichage OpenLayers et l'interface CAP CLAIR.

Sources officielles utilisées pour cette première validation :

- SUP AIP 154/26 - ZRT MAGNAC ALPHA et BRAVO ;
- SUP AIP 150/25 - ZRT SOTAME BRAVO ;
- SUP AIP 145/26 - ZRT BARCARES.

Les géométries ont été retranscrites depuis les limites latérales publiées dans les PDF du SIA. Cette base n'est ni exhaustive ni une source opérationnelle autonome. La validité, les horaires et l'activation réelle doivent toujours être vérifiés auprès du SIA, de SOFIA et des NOTAM.
